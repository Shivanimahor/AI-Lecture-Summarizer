# AI Lecture Summarizer 

An AI-based web application that converts lecture audio/video into:
- Clean Transcript
- Smart Notes
- Summary
- MCQ Quiz

##  Features
- Speech-to-Text using Whisper
- AI-generated Notes & Summary
- Quiz Generation
- PDF Download
- Flask Web App
- User Login & Signup

## Technologies Used
- Python
- Flask
- OpenAI API
- Whisper
- HTML/CSS
- JavaScript
